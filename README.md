# Zona Pizza Web Premium

Sitio web premium y personalizado para Zona Pizza.

## Incluye
- Diseño visual premium inspirado en la línea gráfica real del restaurante
- Carrito funcional
- Upsells
- Delivery / Take out
- Pedido ordenado por WhatsApp
- Menú completo cargado
- Galería con el menú original
- Versión móvil optimizada

## Nota sobre WhatsApp
En `config.js` puedes agregar el número oficial si luego te lo facilitan. Si queda vacío, la web igualmente abre WhatsApp con el mensaje del pedido listo para compartir.
